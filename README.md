# Console Block Blast

A polished Block Blast-inspired game that you can paste directly into the browser console.

## Usage

1. Open any web page.
2. Open DevTools → Console.
3. Paste the contents of `block-blast-console.js` and press Enter.
4. Use **Ctrl+E** to show/hide the panel.

## Included features

- 8×8 board and drag/drop block placement.
- 3-piece hand that refills after all 3 are used (matches core Block Blast loop).
- Row and column clears with combo scoring.
- Best score persistence via `localStorage`.
- Reset button, draggable and resizable game panel, and game-over detection.
- make sure to zoom out if you cant see all the blocks.
